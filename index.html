Os principais comandos para se criar um site

Comandos Básicos de HTML

O HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Alguns dos principais comandos incluem:

<h1> a <h6> - Títulos

Esses elementos são usados para definir os títulos na página, com <h1> sendo o maior e <h6> o menor.

<p> - Parágrafo

O elemento <p> é utilizado para criar parágrafos de texto.

<img> - Imagem

O elemento <img> é utilizado para inserir imagens na página.

Comandos Básicos de CSS

O CSS (Cascading Style Sheets) é utilizado para estilizar páginas web. Alguns dos principais comandos incluem:

Seletores e Propriedades

Os seletores são utilizados para aplicar estilos específicos a elementos HTML. As propriedades definem como esses elementos serão estilizados.

Fontes e Cores

As fontes e cores podem ser definidas utilizando propriedades como font-family, font-size e color.
